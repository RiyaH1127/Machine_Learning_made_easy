# 📊 Multivariate Analysis

## 📌 Overview

This section focuses on **Multivariate Analysis**, where multiple variables are analyzed together to understand relationships, patterns, and dependencies in the dataset.

Unlike univariate analysis (single variable), multivariate analysis helps in identifying how variables interact with each other.

---

## 🎯 Objectives

* Understand relationships between multiple features
* Identify correlations and dependencies
* Detect patterns and trends across variables
* Prepare data for machine learning models
